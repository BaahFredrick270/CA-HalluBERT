# CA-HalluBERT
CA-HalluBERT: A Context-Aware Hybrid Transformer Framework for Detecting Hallucinations in Generative AI Responses Using Local Knowledge Bases — A Case Study of Ghanaian Higher Education.
This repository contains the implementation of CA-HalluBERT, a novel context-aware hybrid transformer framework proposed for detecting hallucinations in generative AI responses within the Ghanaian higher education context. CA-HalluBERT is engineered by fusing a domain-adapted BERT encoder with a locally constructed KNUST knowledge graph, applying a fabricated Cross-Attention Alignment Module (CAAM) to compute alignment scores between AI-generated responses and verified institutional knowledge entries. This work is submitted as part of an ML Research Proposal at KNUST (2026).
